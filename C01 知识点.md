创建空列表

```
a=[]
```

访问列表中元素

```
a[i]
```

列表末尾添加元素

```
a.append(x)
```

获取列表长度（len for length）

```
len(a)
```

遍历列表

```
for i in range(0, len(a)):
	print(a[i])
```

打印完无换行

```
print(东西, end="")
```

换行符号

```
"\n"
```

转义字符

```
"\"
```

```
print("learnhard")
print("le\ar\nhard")
```

函数

```
def functionname(parameter1, parameter2):
	print(parameter1, parameter2)
```
